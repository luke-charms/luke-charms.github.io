Programming III

Functional and Implementational Content

Functional Content is to simply take an input value n and return an output value “sum of n squares”

Implementational Content says how this should be done (create locations in memory, increment one iteratively, etc…)

Functional Programming contains two aspects in their specification:

  • High level behaviour and low level implementation
  • Humans better at former!

Key Idea of Func. Programming

  • Concentrate on high level “functional” behaviour
  • Leave memory management and implementation details to the compiler
sumsq n = sum [x^2 | x <- [1..n]]

We have defined a function called sumsq, we can “run” the program by simply applying the function

What is it (functional programming) ?

A programming style !

Basic method of computation is the application of functions to arguments

Functional programming language:

Languages designed to support and encourage the functional programming style
