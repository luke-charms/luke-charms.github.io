How is an image formed?

Decomposing an image into bits

Every pixel is an 8-bit unsigned integer in (0,255)

Colour is represented by a vector stored in the pixel (e.g., using RGB for storing an image results in a 3x1 matrix stored at every bit)

The Most Significant Bit carries the most information whereas bit 0 is noise

Effects of differing image resolution

Low resolution loses information but NxN points implies a lot of storage - so how to choose an appropriate value for N? (example resolutions: 64x64; 128x128; 256x256)

Frequency

The frequency of a wave is the number of waves passing a point in a certain time. We normally use a time of one second, so this gives frequency the unit hertz, since one hertz is equal to one wave per second.

The relationship between the frequency 𝜉 and the period 𝑇 is: 𝜉 = 1/𝑇

Fourier said…

Any periodic function is the result of adding up sine and cosine waves of different frequencies
