How can we improve first order edge detection?

Applying Sobel operator

Sobel is a good basic operator

Canny edge detection operator

Formulated with three main objectives:

  • optimal detection with no spurious responses
  • good localisation with minimal distance between detected and true edge position - i.e., one pixel width is enough to detect an edge, don’t need more!
  • single response to eliminate multiple responses to a single edge

Approximation

  1. use Gaussian smoothing;
  1. use the Sobel operator; (can combine both of these)
  1. use non-maximum suppression; and
  1. threshold with hysteresis to connect edge points.
