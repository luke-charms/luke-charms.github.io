Scenarios

Scenario

User opens software (tool)

Input page is displayed

User can drag and drop the relevant three CSV files into each of the input areas of the screen to start processing the data

When the user has dropped each CSV file, the relevant metrics and data corresponding to that file gets updated, displayed and a green checkmark or red ‘X’ is either shown, depending on if the data was in the correct format or not

User can navigate to the Dashboard or Graph page by selecting their tab

Scenario 1 - John

John opens the software

Data Input page is displayed

John drags his three separate CSV files and drops each of them in the correct place in the tool

Confirmation is shown that the data is in the correct format and place

John navigates to the Graph page by selecting the tab

John views the displayed charts from his imported data and changes time granularity to match his preferred period of time to view the graphs from

John filters the graphs by date range to narrow his search and specifically view only a brief period of the ad campaign

John can select each individual metric to individually compare the different sections of the ad campaigns to get precise information on how the campaign is going

Scenario 2 - Mark

Mark opens the software

Data Input page is displayed

Mark drags in the Impression Log CSV file and the Click Log CSV file and drops them in the first and second bar of the tool

Confirmation is shown that the data is in the correct format and place

Mark navigates to the Dashboard page first, by selecting the tab

Mark views the relevant calculated data, that can be generated based on his CSV files, and writes down the important figures he needs to remember

Mark then navigates to the Graph page by selecting the tab

Mark views the displayed histogram chart of the click costs and filters his campaign by audience segments to get a better understanding of his data

Scenario 3 - Anna

Anna opens the software

Data Input page is displayed

Anna drags in the Server Log CSV file into the second input slot

An error message is displayed saying that the data is in the incorrect slot

Anna drags again the Server Log CSV file into the thrid input slot instead

Confirmation is shown that the data is in the correct format and place

Anna navigates to the Graph page by selecting the tab

Anna slides the time granularity slider to adjust the time in which she wants to view the campaign metrics across

Anna navigates to the file tab and clicks to generate a PDF of the data shown in the graph on the screen

Scenario 4 - Jim

Jim opens the software

Data Input page is displayed

Jim drags and drops his three CSV files and places them in the correct input areas

Confirmation is shown that the data is in the correct format and place

Jim navigates to the Dashboard page by selecting the tab

Jim views the overall campaign performance metrics and compares them to previous campaigns he has run

Jim defines in the editor how he wants to register a bounce and views the updated data

Jim filters the metrics by context to see how the campaign is performing in different areas of public interest

Jim decides to adjust his campaign strategy and navigates back to the Data Input page to upload new CSV files with different data

Jim drags and drops the new CSV files and the relevant metrics and data corresponding to that file gets updated and displayed on the Dashboard page

Scenario 5 - Judy

Judy opens the software

Data Input page is displayed

Judy drags and drops her Click Log and Server Log SQLite files into the second and third column of the software

An error is generated saying the data is in the wrong format

Judy drags and drops her Click Log and Server Log re-generated CSV files into the second and third column of the software

Confirmation is shown that the data is in the correct format and place

Judy navigates to the Graph page by selecting the tab

Judy filters the graphs by date range; audience segments and context to see how the campaign is performing on all areas of the public

Judy adjusts the time range to compare the performance of the campaign in different weeks

Judy clicks on the "Export to PDF" button to save the data shown in the graphs as a PDF file for further analysis with her co-workers
