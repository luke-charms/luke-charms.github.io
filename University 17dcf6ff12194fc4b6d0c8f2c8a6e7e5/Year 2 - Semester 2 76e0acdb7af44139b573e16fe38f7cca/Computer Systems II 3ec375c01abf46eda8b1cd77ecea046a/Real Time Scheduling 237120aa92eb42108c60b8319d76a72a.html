CPU Utilisation

Concerned with whether out task set has enough CPU ___ to run

U = C total - C idle ≤ 1

Ctotal → Total CPU time available (100%)

Cidle → Fraction of CPU time spent in idle task or sleeping

Assumption for Analysis

1. Tasks are periodic

→ convert aperiodic tasks by polling

2. The deadline for a task is its next invocation

3. Context switches take no time

→ leave some margin in duration and deadline

→ context switch should be fast to accommodate for a fixed margin (worst case switches which could appear)
